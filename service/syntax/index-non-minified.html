An online tool to display formatted source code

  • No dynamic styling (no FOUC - Flash Of Unstyled Content)
  • Small footprint - no JavaScript brush to load (JS + CSS is about 6kb)
  • Users can open the code in a popup window for easy printing
  • Users can swap between formatted and non-formatted views
  • Users can collapse and expand large snippets
  • Users can navigate the options via the keyboard
  • Developers can choose to emphasize arbitrary line(s)
  • Developers can turn line numbering on or off
  • URLs inside formatted snippets are click-able
